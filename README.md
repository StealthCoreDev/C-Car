# StopCarApp

این فایل موقتی در این صفحه‌ی گیت‌هاب متعلق به **Artin Ahmadpoori** است و برای درس «توسعه برنامه‌سازی و پایگاه داده» نوشته شده است.

## برنامه چکار می‌کند؟
یک ماشین دو‌بعدی ساده که با نگه داشتن یک دکمه به سمت یک مخروط ایست حرکت می‌کند و نزدیک آن خودکار متوقف می‌شود. با رها کردن دکمه نیز بلافاصله می‌ایستد.

## چطور نصب شود؟
فایل `StopCarApp_v1.0.0_Setup.exe` را اجرا کنید و مراحل نصب را دنبال کنید. بعد از نصب، برنامه از روی دسکتاپ یا منوی Start قابل اجراست.
